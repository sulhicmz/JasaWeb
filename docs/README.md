## Project Overview

JasaWeb is a comprehensive web development service platform that helps businesses manage client relationships, deliver website projects efficiently, and provide transparent collaboration tools.

- **Technology Stack**: Astro, NestJS, PostgreSQL, Prisma, Tailwind CSS
- **Architecture**: Monorepo structure with workspace management
- **Target Markets**: School Websites, News Portals, Company Profiles
- **Core Features**: Client Portal, Project Management, File Sharing, Approval Workflows, Ticket System

For more information, see the main repository [README.md](../README.md).
